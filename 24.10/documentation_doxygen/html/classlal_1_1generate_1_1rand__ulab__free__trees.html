Detailed Description

Uniformly random selection of unlabelled free trees.

This is a wrapper class of _rand_ulab_free_trees. Users should refrain from using said class. However, users will find the implementation details (as for algorithms and citations) in the documentation of said class.

An example of usage of this class is

lal::generate::rand_ulab_free_trees Gen(n);
for (int i = 0; i < 100; ++i) {
const lal::graphs::free_tree T = Gen.get_tree();
// ...
}
