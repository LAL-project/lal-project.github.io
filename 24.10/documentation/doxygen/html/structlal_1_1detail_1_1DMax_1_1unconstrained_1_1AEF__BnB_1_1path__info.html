Detailed Description

Data related to paths in the tree.

This struct contains:

  • Counting of vertices assigned to the arrangement,
  • Maximum and minimum number of vertices of level values \(\pm2\) (these are generous bounds)
  • Actual maximum and minimum number of vertices of level values \(\pm2\).
