Detailed Description

A class to represent a coloring of the vertices of a bipartite graph.

Coloring of general graphs is known to be NP-complete [33]. This class is used to represent the vertex coloring of bipartite graphs of n vertices, for which two colors are sufficient. This class uses static members red and blue to designate these colors. Function lal::properties::bipartite_coloring calculates such coloring.
