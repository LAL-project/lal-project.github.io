Detailed Description

template<class T, bool unique, typename allocator_t = std::allocator<T>>
class lal::detail::sorting::sorted_vector< T, unique, allocator_t >

Sorted vector class.

This class inherits from std::vector to implement a vector-like container whose elements are always sorted lexicographically non-decreasing.

Template Parameters
Ttype of data.
uniqueWhether or not the elements inserted should be unique.
